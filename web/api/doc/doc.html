Голосарий

Понятие Описание
Клиент Банк, кредитор. Кредиторов много. У каждого кредитора есть перечень кредитных договоров (дел).
Дело Кредитный договор (контакты, сумма, срок и т. д.). Дело заёмщика не выполняющего обязанности кредитного договора обрабатывается опереторами с помощью звонков, а так же автоматической рассылкой бумажных писем, смс и звонков IVR на основе созданных стратегий.
Дело активное Статус дела. Дело со статусом "активное" находится в работе
Контакт Адресат уведомлений. Контакт - часть дела. У дела может быть несколько контактов.
Стратегия Перечень активностей выполняемых за указанный период времени. Стратегия привязана к клиенту и применяется к делам клиента. Клиент может быть привязан к нескольким стратегиям. Также, у одной стратегии может быть несколько клиентов. Привязка клиентов к стратегиям осуществляется посредством фильтров стратегии.
Ручная стратегия Стратегия, которая привязана к клиенту, либо к рееестру клиентов. Задачи, которые генерируются этой стратегией выполняются в любом случае (даже, если задача повторяется в рамках одного дня). В активностях ручной стратегии указывается шаблон, по которому будут созданы автоматический задачи.
Основная стратегия Стратегия, которая не привязана к клиенту, либо к рееестру клиентов. Привязка стратегии к клиенту происходит на странице клиента. Одна основная стратегия может быть привязана к нескольким контрагентам. Задачи, которые генерируются основными стратегиями в рамках одного дня НЕ должны повторятся.
Тактика / активность алгоритм уведомлений. Тактика - часть определенной стратегии. Тактику определяют: инструмент, график, фильтр.
Инструмент орудие реализации тактики. Инструмент применяется к делу. Инструмент является частью тактики стратегии.
График график отправки задач в систему ProCollect. График формируется на основе фильтров "Дни"
Фильтр Условие отбора дел. Каждая тактика применяется к делам отфильтрованным по ряду характеристики. Перечень характеристик для отбора дел - часть тактики.
Шаблон шаблон уведомления. Для определения шаблона используется клиент + инструмент.
